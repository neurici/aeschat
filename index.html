About AESchat

Features:
- Fast and easy way to securely chat
- End-to-end encryption
- Encryption key derived from the room name
- No message stored on the server
- Webpage delivered over HTTPS/SSL
- Single HTML file, source at GitHub, view source
- Uses bcrypt, SHA256 and AES
- Accessible from TOR (url)
- Easy to generate link addresses
